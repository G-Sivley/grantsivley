Med Buddy

Description

Med Buddy is an iOS application with the goal of making medication adherence a part of life. It is an application with the ability to save your daily medications, schedule alerts, and share your medications with your providers.

Technologies I Used

Firebase - To save consumers medication information for presentation within the app
Xcode/Swift - The IDE and language used for iOS applications. I worked on my skill with storyboards/iOS UI, delegate methods, and the MVC model.
Calendar and Image Picker - I learned how to create a satisfing UI with a calendar attached to the backend with Firebase.

Stock Screener

Description

StockScreener is a Python application that uses Selenium, BeautifulSoup, time, requests, and webbrowser to open multiple pages of information regarding stock tickers that return on my Stock filter. It opens as many pages as the user requests in one click. This has saved time for me allowing me to research each stock with ease.

Technologies I Used

Selenium - I have learned how to use Selenium to search HTML on a page and pull multiple links from a page.
BeautifulSoup - Parse HTML to find the correct values and apply them to a variable that can be used with webbrowser.
VIM - This is my first time using VIM and it has significantly improved my coding speed.

This Website

Description

This was initially a way for me to show off my projects. I have learned to appreciate the complexities of creating, hosting, and running a website. Through this experience, I have had to learn multiple technologies that I was not comfortable with in the past.

Technologies I Used

HTML/CSS/JavaScript - I had to learn the main "languages" for creating websites to complete this project.
Flask and Jinja2 - Using Python, I have learned the basic GET/POST requests, using a basic backend server. Also, I have also learned to create efficent code with Jinja2. This allows me to reuse HTML snippets all over the website.
Heroku - This is my first time using a server for dynamic websites. I learned how to deploy a website with heroku with no charge to me (besides the domain name).
Ajax and Mailgun - Using Ajax provided by jquery, I learned how to use async JS POST methods to send emails to myself without refreshing the page.
